# **How we keep it fun (and fair)**

Short milestones over \~2-week windows. We agree what “done” looks like before we start — so nobody’s surprised, nobody resents anyone, and we’re all on the hook for a minimum bit of progress. Want to wind down after a milestone’s met? Allowed, not a betrayal.

# **How we make decisions**

* **Lane owner has the final call** in their domain. Decide fast — don’t turn everything into a committee.

* Not the owner? **Advise hard, then disagree-and-commit.**

* **One exception — “one-way doors.”** Anything hard or costly to reverse (platform, data model, core architecture, auth — anything that changes what the others build on) gets a quick thumbs-up from all three before it’s locked. Reversible stuff, the owner just does.
